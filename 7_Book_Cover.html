9 Billion Names

Tool: Adobe Illustrator

This movie poster is inspired by the book "The Nine Billion Names of God," where monks of a mountain monastery seek to calculate the true name of God using a computer. This fictional poster was created only using Adobe Illustrator. I was never big into graphic design nor vector graphics so this was a real challenge for me.
