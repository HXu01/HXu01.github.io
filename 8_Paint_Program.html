Drawing Program

Tool: Visual Studio Code, P5.js

A drawing program I wrote using JavaScript. This program features five different tools with five available colors. Each tool is activated with the corresponding number key, and each color uses the corresponding button. The pen size can be changed using the '[' and ']' keys, with the size number displayed in the 'size' box. I wrote this program with the intention of creating a tool that can produce comic-style drawings and paintings, as reflected in the pen types and color palette. All drawings below were created using this program.

Over all this was a very challenging project. It was my first real attempt at writing a program using js. I learned a lot from this program, and I'm excited to apply the knowledge gained here to future projects.
